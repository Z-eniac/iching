엇갈림과 불화, 차이의 인정

38 睽규

“서로 어긋나 시선이 갈리는 때.
다름을 인정하고, 진솔하게 마주하라.”

🔸키워드

• 엇갈림·불화 · 차이관리 · 다양성 · 조율·협상 · 비동의·이견 · 작은일부터 · 디자인싱킹 · 관계재설정

🔹원문 인용 & 풀이

• 卦辭괘사

「睽：小事吉.」

“엇섬—작은 일은 길하다.”

• 彖曰단왈

「睽, 外文明而內悅, 柔進而上行. 睽之時用大矣哉！」

“겉은 밝고 속은 기쁨—부드러움이 위로 나아간다. 엇섬의 때를 잘 쓰면 크다.”

• 象曰상왈

「上火下澤, 睽. 君子以同而異.」

“위에는 불, 아래엔 못—군자는 ‘같음 속의 다름’을 살린다.”

🔹메타 정보

• 기본 정보 : 上離 [ 火 / 南 / 中女 / 目·心 ] / 下兌 [ 金 / 西 / 少女 / 口·肺 ]

• 짝괘 : 錯착괘 = No.39 / 綜종괘 = No.37 / 互호괘 = No.63

• 정위: 1

• 응/불응 : 1–4 불응 / 2–5 응 / 3–6 응

🔹효사 핵심

• 1효: “길을 잃은 작은 모습” — 찾아오면 길.

• 2효: “수레의 멍에가 벗어남” — 임시 분리.

• 3효: “눈이 서로 어긋남” — 중재가 필요.

• 4효: “홀로 맞섬” — 위태하나 바르면 길.

• 5효: “한집에 같이 살되 마음이 다름” — 정직한 소통으로 길.

• 6효: “큰 엇섬” — 처음은 흉이나 끝내 길.
