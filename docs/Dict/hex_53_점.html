53 漸점

“기러기가 물에서 언덕으로 오르는 때.
서두르지 말고, 예로써 차례를 밟아라.”

🔸키워드

• 점진·단계 · 성숙·혼인 · 장기성장 · 습관형성 · 학습곡선 · 인내 · 절차·예식 · 정착

🔹원문 인용 & 풀이

• 卦辭괘사

「漸：女歸吉, 利貞.」

“점진—여인의 시집감이 길하다. 바름이 이롭다(서서히 성숙).”

• 彖曰단왈

「漸, 進也. 女歸吉, 利貞；進得位, 往有功也. 進以正, 可以正邦也.」

“점차 나아감. 바르게 자리를 얻어 가면 공이 생기고, 바른 전진은 나라까지 바로잡을 수 있다.”

• 象曰상왈

「山上有木, 漸. 君子以居賢德, 善俗.」

“산 위에 나무—군자는 어진 이를 모시고 풍속을 선하게 한다.”

🔹메타 정보

• 기본 정보 : 上巽 [ 木 / 東南 / 長女 / 股·髮 ] / 下艮 [ 土 / 東北 / 少男 / 手·背 ]

• 짝괘 : 錯착괘 = No.54 / 綜종괘 = No.54 / 互호괘 = No.64

• 정위: 2·3·4·5

• 응/불응 : 1–4 불응 / 2–5 응 / 3–6 불응

🔹효사 핵심

• 1효: “기러기가 물가에” — 작은 전진, 길.

• 2효: “바위에” — 곧 터를 잡음.

• 3효: “육지에” — 멀지 않아 장가감, 길.

• 4효: “나무 위에” — 가구를 얻고 공을 이룸.

• 5효: “언덕에” — 길. 끝내 이로움.

• 6효: “높은 언덕 위에” — 기세가 과하면 흉, 겸손 유지.
