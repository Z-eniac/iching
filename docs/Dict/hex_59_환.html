59 渙환

“흩어짐이 모임의 씨가 되는 때.
마음을 씻고, 사당을 세워 뜻을 모아라.”

🔸키워드

• 해빙·해소 · 분산→결집 · 의례·공명 · 대규모이동 · 크라우드매니지 · 시스템재부팅 · 분규해결 · 네트워크

🔹원문 인용 & 풀이

• 卦辭괘사

「渙：亨. 王假有廟, 利涉大川, 利貞.」

“흩어짐·해산—형통. 임금이 종묘에 나아감이 좋고, 큰 강을 건너도 이롭다. 바름이 이롭다.”

• 彖曰단왈

「渙, 亨. 剛來而不窮；柔得位乎外而上同. 王假有廟, 致孝享也.
利涉大川, 乘木有功也. 利貞, 乃應乎天也.」

“강이 와도 막힘이 없고, 부드러움이 제자리에 올라 합한다.
종묘의 예로 뜻을 모으고, 배(木)로 강을 건너 공을 세운다—바름으로 하늘에 응한다.”

• 象曰상왈

「風行水上, 渙. 先王以享于帝, 立廟.」

“바람이 물 위로 불어 흩어짐—선왕은 상제를 제사하고 사묘를 세워 마음을 모았다.”

• 기본 정보 : 上巽 [ 木 / 東南 / 長女 / 股·髮 ] / 下坎 [ 水 / 北 / 中男 / 耳·腎 ]

• 정위: 4·5

• 응/불응 : 1–4 불응 / 2–5 불응 / 3–6 응

🔹효사 핵심

• 1효: “튼튼한 말을 써 구휼” — 길.

• 2효: “흩어져 달아남” — 후회가 사라짐(재정렬).

• 3효: “자신의 집착을 흩어냄” — 후회 없음.

• 4효: “무리를 흩어 모으니 크게 길” — 구릉처럼 모임.

• 5효: “큰 호령이 흩어지니 왕이 거처를 옮김” — 허물 없음(대전환).

• 6효: “피를 흩어 멀리 떠남” — 허물 없음(재앙에서 벗어남).
