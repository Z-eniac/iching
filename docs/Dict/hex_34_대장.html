34 大壯대장

“힘이 크게 일어나는 때.
바르게 쓰고, 성급한 과시는 금하라.”

🔸키워드

• 기세·파워 · 확장·공세 · 원칙준수 · 속도관리 · 권위와예 · 타이밍 · 힘의절제 · 윤리

🔹원문 인용 & 풀이

• 卦辭괘사

「大壯：利貞.」

“크게 장성—바름이 이롭다.”

• 彖曰단왈

「大壯, 大者壯也. 剛以動, 故壯. 大壯利貞, 正大而天地之情可見也.」

“강함이 움직여 기세가 왕성하다—크되 바름을 지켜야 한다.”

• 象曰상왈

「雷在天上, 大壯. 君子以非禮弗履.」

“하늘 위에 우레—군자는 예가 아니면 밟지 않는다.”

🔹메타 정보

• 기본 정보 : 上震 [ 木 / 東 / 長男 / 足 ] / 下乾 [ 金 / 西北 / 父 / 首·骨 ]

• 짝괘 : 錯착괘 = No.20 / 綜종괘 = No.33 / 互호괘 = No.43

• 정위 : 1·3·6

• 응/불응 : 1–4 불응 / 2–5 응 / 3–6 응

🔹효사 핵심

• 1효: “강력하되 집을 지키면 허물 없다.”

• 2효: “바퀴살이 부러진 수레” — 무리한 속도 경계.

• 3효: “작은 염소가 울타리를 들이받음” — 무모한 돌파는 흉하다.

• 4효: “수양하여 때를 기다리면 허물 없다.”

• 5효: “양의 뿔이 하늘에 닿음” — 과시를 경계하라.

• 6효: “울타리를 넘어섬” — 후회와 허물; 의례를 넘지 말라.
