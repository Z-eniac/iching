46 升승

“나무가 땅을 뚫고 오르는 때.
성실히 한 계단씩, 조용히 올라라.”

🔸키워드

• 승진·등용 · 점진상승 · 멘토십 · 남방(공개/양성) · 장기축적 · 순응의오름 · 계단식성장 · 낙관

🔹원문 인용 & 풀이

• 卦辭괘사

「升：元亨. 用見大人, 勿恤.南征吉.」

“오름—크게 형통. 대인을 뵈라, 근심 말라. 남쪽으로 나아가면 길하다.”

• 彖曰단왈

「升, 柔以時升, 巽而順, 剛中而應, 是以大亨. 用見大人, 勿恤；有慶也.南征吉, 志行也.」

“부드러움이 때에 맞춰 오름—공손과 순응, 가운데의 바름으로 응해 크게 형통.
대인을 만나 뜻을 행하라.”

• 象曰상왈

「地中生木, 升. 君子以順德, 積小以高大.」

“땅속에서 나무가 돋아남—군자는 덕을 따라, 작음을 쌓아 크게 높인다.”

🔹메타 정보

• 기본 정보 : 上坤 [ 土 / 西南 / 母 / 腹·肌肉 ] / 下巽 [ 木 / 東南 / 長女 / 股·髮 ]

• 짝괘 : 錯착괘 = No.25 / 綜종괘 = No.45 / 互호괘 = No.54

• 정위: 3·4·6

• 응/불응 : 1–4 불응 / 2–5 응 / 3–6 응

🔹효사 핵심

• 1효: “참되게 오른다” — 크게 길하다.

• 2효: “성실하면 간소제도 이롭다” — 허물 없음.

• 3효: “빈 고을로 오른다” — 빈터부터 기반을 쌓아라.

• 4효: “왕이 기산에서 제사를 드림” — 길하고 허물 없다.

• 5효: “바르고 길” — 계단을 오른다.

• 6효: “어둠 속의 오름” — 쉬지 않는 바름이 이롭다.
