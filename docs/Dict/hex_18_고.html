18 蠱고

“누적된 폐단을 씻어내야 할 때.
뿌리부터 바로잡고, 집안부터 정돈하라.”

🔸키워드

• 정화·수습 · 레거시개혁 · 사전·사후점검 · 리팩토링 · 위생·클린업 · 거버넌스수리 · 대공사준비 · 문화재건

🔹원문 인용 & 풀이

• 卦辭괘사

「蠱：元亨, 利涉大川. 先甲三日, 後甲三日.」

“크게 형통하고 큰 강을 건너는 것이 이롭다. 갑일 사흘 전과 사흘 뒤(사전·사후 정비).”

• 彖曰단왈

「蠱, 剛上而柔下, 巽而止, 蠱. 蠱元亨, 而天下治也. 利涉大川, 往有事也.
先甲三日, 後甲三日, 終則有始, 天行也.」

“부패를 고치고 질서를 재정비하는 때. 큰일을 위해 사전·사후 점검을 철저히.”

• 象曰상왈

「山下有風, 蠱. 君子以振民育德.」

“산 아래 바람—군자는 백성을 일으키고 덕을 기른다.”

🔹메타 정보

• 기본 정보 : 上艮 [ 土 / 東北 / 少男 / 手·背 ] / 下巽 [ 木 / 東南 / 長女 / 股·髮 ]

• 짝괘 : 錯착괘 = No.17 / 綜종괘 = No.17 / 互호괘 = No.54

• 정위: 3·4

• 응/불응 : 1–4 불응 / 2–5 응 / 3–6 불응

🔹효사 핵심

• 1효: “부모의 허물을 바로잡되, 아들로서 바르게 하면 허물 없다.”

• 2효: “장자의 허물을 고치면 길하다.”

• 3효: “부모의 허물을 지나치게 탓하면 뉘우침” — 온정과 원칙의 균형.

• 4효: “스스로 허물을 고치면 허물 없다.”

• 5효: “임금의 허물을 고치면 길하고 후회 없다.”

• 6효: “새로운 일을 시작하면 흉” — 먼저 정비가 우선.

🔹적용 가이드
