단호한 결단의 선포

43 夬쾌

“결단이 하늘 뜰에 드러나는 때.
칼을 휘두르지 말고, 바름으로 알리라.”

🔸키워드

• 결단·단절 · 공표·선언 · 리스크관리 · 무력·강행금지 · 분배·환원 · 명분 · 준법·정절 · 후폭풍관리

🔹원문 인용 & 풀이

• 卦辭괘사

「夬：揚于王庭, 孚號, 有厲. 告自邑, 不利即戎；利有攸往.」

“결단—왕정에서 드높이 알리되, 믿음의 외침이 있어도 위태롭다.
읍에서 알림이 옳고, 바로 군사로 나아감은 불리; 나아감이 이롭다.”

• 彖曰단왈

「夬, 決也.剛決柔也. 健而說, 決而和. 揚于王庭, 柔乘五剛也. 孚號有厲, 其危乃光也.
告自邑, 不利即戎, 所尚乃窮也. 利有攸往, 剛長乃終也.」

“결단하여 부드러움을 끊음. 건(乾)과 열(兌)이 합해 결단하되 화합을 지향.
공표는 함부로 군사화하지 말고, 정정당당히 진행.”

• 象曰상왈

「澤上于天, 夬. 君子以施祿及下, 居德則忌.」

“하늘 위에 못—군자는 녹을 아래에 베풀고, 공덕에만 안주하길 꺼린다.”

🔹메타 정보

• 기본 정보 : 上兌 [ 金 / 西 / 少女 / 口·肺 ] / 下乾 [ 金 / 西北 / 父 / 首·骨 ]

• 짝괘 : 錯착괘 = No.23 / 綜종괘 = No.44 / 互호괘 = No.01

• 정위: 1·3·5·6

• 응/불응 : 1–4 불응 / 2–5 불응 / 3–6 응

🔹효사 핵심

• 1효: “발가락에 힘이 앞서면 감당 못해 허물이 생긴다.”

• 2효: “두려움의 외침” — 밤에 적이 있어도 근심하지 말라.

• 3효: “광대뼈의 힘” — 흉. 군자는 독행의 결단, 비에 젖어도 성냄 없이 가면 허물 없다.

• 4효: “엉덩이에 살이 없다” — 나아감이 더딤. 양을 끌면 후회가 없고, 말을 들어도 믿지 않음.

• 5효: “비름같이 무성히 끊어냄” — 가운데 길로 가면 허물 없다.

• 6효: “외침 없이 끝내 흉하다” — 경고를 무시하면 화.
