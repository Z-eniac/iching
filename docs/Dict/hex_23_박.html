한 줄 요지
깎여 나감, 쇠퇴 속 전환 준비

23 剝박

“껍질이 벗겨져 허실이 드러나는 때.
낡은 것을 버리고, 근본을 보전하라.”

🔸키워드

• 퇴각·보전 · 축소·절약 · 구조침식 · 리스크차단 · 핵심보호 · 내부정비 · 저점관리 · 겸양

🔹원문 인용 & 풀이

• 卦辭괘사

「剝：不利有攸往.」

“벗겨지고 깎이는 때—나아감이 이롭지 않다.”

• 彖曰단왈

「剝, 柔變剛也. 不利有攸往. 小人長也. 順而止之, 觀象剝也. 剝之時義大矣哉！」

“유약함이 강함을 침식한다. 소인이 자라고 군자가 물러나는 때—멈추며 보전하라.”

• 象曰상왈

「山附於地, 剝. 上以厚下, 安宅.」

“산이 땅에 붙은 상—윗사람은 아랫사람을 두텁게 하여 집을 편안히 한다.”

🔹메타 정보

• 기본 정보 : 上艮 [ 土 / 東北 / 少男 / 手·背 ] / 下坤 [ 土 / 西南 / 母 / 腹·肌肉 ]

• 짝괘 : 錯착괘 = No.43 / 綜종괘 = No.24 / 互호괘 = No.02

• 정위: 2·4

• 응/불응 : 1–4 불응 / 2–5 불응 / 3–6 응

🔹효사 핵심

• 1효: “깔개가 벗겨진다” — 작은 것부터 무너짐. 바로잡으면 길.

• 2효: “귤나무가 변해 탕나무” — 환경에 따라 변질됨을 경계.

• 3효: “벗겨지는 것이 더 심해진다” — 중지하고 보전.

• 4효: “기둥이 깎인다” — 집의 뼈대를 보호하라.

• 5효: “바닥의 깔개가 벗겨짐” — 최저점을 직면하라.

• 6효: “대도(大道)가 끊어진 듯” — 군자라면 집에 거하며 명을 기다려라.
