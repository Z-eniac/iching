기쁨과 열린 교류

58 兌태

“못이 서로 이어져 기쁨이 흐르는 때.
즐거움에 바름을 더하고,
헛된 웃음을 경계하라.”

🔸키워드

• 즐거움·설득 · 커뮤니티 · 학습·토론 · 톤·분위기 · 내강외유 · 동기부여 · 문화·사기 · 윤리적낙관

🔹원문 인용 & 풀이

• 卦辭괘사

「兌：亨, 利貞.」

“기쁨—형통, 바름이 이롭다.”

• 彖曰단왈

「兌, 說也. 剛中而柔外, 說以利貞. 乃順乎天而應乎人.」

“기쁨과 설득의 도—안은 강건·중정, 겉은 부드러워 바름을 이롭게 한다.
하늘에 순응하고 사람에 호응한다.”

• 象曰상왈

「麗澤, 兌. 君子以朋友講習.」

“연못이 서로 이어짐—군자는 벗과 함께 토론하고 익힌다.”

🔹메타 정보

• 기본 정보 : 上兌 [ 金 / 西 / 少女 / 口·肺 ] / 下兌 [ 金 / 西 / 少女 / 口·肺 ]

• 짝괘 : 錯착괘 = No.52 / 綜종괘 = No.57 / 互호괘 = No.37

• 정위: 1·5·6

• 응/불응 : 1–4 불응 / 2–5 불응 / 3–6 불응

🔹효사 핵심

• 1효: “온화한 기쁨” — 길하다.

• 2효: “성실한 기쁨” — 길하여 후회 없음.

• 3효: “찾아오는 얄팍한 즐거움” — 흉(경박 경계).

• 4효: “장사치의 말재주” — 불안정하나 경계하면 기쁨.

• 5효: “벗겨내듯 기쁨” — 위태(기쁨의 남용 경계).

• 6효: “끌어내는 기쁨” — 경계: 유혹·선동 조심.
