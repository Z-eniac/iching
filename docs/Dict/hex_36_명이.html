36 明夷명이

“빛이 땅속으로 잠기는 때.
겉빛을 감추고, 속빛을 지켜라.”

🔸키워드

• 위기·잠행 · 빛감추기 · 내공·문덕 · 위장·보존 · 생존전략 · 리더십절제 · 난국관리 · 안전

🔹원문 인용 & 풀이

• 卦辭괘사

「明夷：利艱貞.」

“빛이 상함—어려움 속 바름이 이롭다.”

• 彖曰단왈

「明入地中, 明夷. 內文明而外柔順, 以蒙大難, 文王以之. 利艱貞, 晦其明也.」

“빛이 땅속으로 들어간 상—문덕을 간직하고 겉으로는 순하여 큰 난을 이겨낸다.
어려움 속의 바름을 지켜 빛을 감춘다.”

• 象曰상왈

「明入地中, 明夷. 君子以蒞眾, 用晦而明.」

“빛이 땅속으로—군자는 많은 이를 거느릴 때, 겉을 어둡게 하여 속으로 밝게 한다.”

🔹메타 정보

• 기본 정보 : 上坤 [ 土 / 西南 / 母 / 腹·肌肉 ] / 下離 [ 火 / 南 / 中女 / 目·心 ]

• 짝괘 : 錯착괘 = No.06 / 綜종괘 = No.35 / 互호괘 = No.40

• 정위 : 1·2·3·4·6

• 응/불응 : 1–4 응 / 2–5 불응 / 3–6 응

🔹효사 핵심

• 1효: “새의 날개가 상함” — 움직임을 줄여라.

• 2효: “행낭에 상처” — 중상, 정지 필요.

• 3효: “남방의 사냥” — 큰 수확, 그러나 조심.

• 4효: “들보 아래에 숨어 빛을 감춤” — 길하다.

• 5효: “문명한 군자가 상함” — 바르게 견디면 길.

• 6효: “왕이 사냥으로 종을 얻음” — 때를 알면 길.

🔹적용 가이드

✦ 겉은 낮추고 속은 정비(스텔스 모드)

✦ 안전·보안·법무 최우선

✦ 명예·노출 자제, 실력 축적

✦ 난국의 ‘바른 고집’ 유지
